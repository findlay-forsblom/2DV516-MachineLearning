Datasets for Lecture 3 about regression

RealEstate.csv
===============
Observations: 200
Response: price (dollars) , column 1
Features
- Number of rooms, column 2
- Number of bathrooms, column 3
- Area (square feet), column 4

stat_females.csv
================
Observations: 214
Response: a girls height (inches), column 3
Features
- Mom height (inches), column 1
- Dad height (inches), column 2

polynomial.csv
================
Observations: 1000
Response: y, column 2
Feature: x, column 1
Generated as y = 5 + 12*x - x^2 + 0.025*x^3 +normrnd(0,5)
